A Step By Step Procedure to Graphical Password Using Image Segmentation

: Fill the Registration Form consisting of Details like Username, Password, Gender, Address and others. Also, keep a note of the Username and Password.

: Login into the Application and Upload an Image as your Graphical Password.

: The Image Analyst will check the qualit of Image and will fragment the Image into several other Images. You can view the Fragmented Images.

: While you are logging into the Application for the second time, The Fragmented Images will appear on your screen in a Jumbled Manner.

: Arrange the Images in a proper order and then submit the Image. The system will match the Image with the Original Image. If it matches completely, then you will be authenticated to use the Application               otherwise you will be asked to attempt login again.
